## Syntax

### Variables

*`specifier`* *`type`* *`name`* `=` *`value`* 

Example

```c
int number = 69;
```

### Constants

`const` *`type`* *`name`* `=` *`value`* 

Example

```c
const double PI = 3.14159;
```

### Functions

*`type`* *`name`* `(` *`params...`* `)` `{ ... }`

Example

```c
float area(float width, float height) {
    /* code */
    return value;
}
```
